cpp_03 - Inheritance
cpp_04 - Subtype polymorphism, abstract classes, interfaces
cpp_05 - Repetition and Exceptions
cpp_06 - C++ Casts
cpp_07 - C++ Templates
cpp_08 - Templated containers, iterators, algorithms
